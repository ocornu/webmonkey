Class File


Extends nsIFile.
The File class allows to manage files easily. It is centered around simple actions, in particular full-content reading and writing.
Technically, File inherits from nsIFile.
Defined in: file.js.

void load(nsIURI|String aURI, [Function onComplete])
Asynchronously load a target URI in this file.

If present, onComplete is called on completion, with three parameters:
  • nsIChannel : The channel used to load the target URI. If the HTTP protocol is used, this is a nsIHttpChannel.
  • int : The request status code (0 on success). In case of error, this is a HTTP error code if HTTP was used.
  • string : The message associated with this status/error code.
This method bypasses the files cache and replaces existing files. It uses a nsIWebBrowserPersist to perform its task.
