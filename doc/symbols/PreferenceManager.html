Class PreferenceManager

Allow storage and retrieval of (key, value) pairs across tabs, windows and sessions.

  • Keys must be of type string. They are organized in a tree fashion, the leaf/branch separator being a dot (see about:config for an example).
  • Values must be of type string, boolean or integer (i.e. a Number without decimal part, between PreferenceManager.MIN_INT_32 and PreferenceManager.MAX_INT_32).
This simple API sits on top of nsIPrefService.
Defined in: prefmanager.js.
